Flask Starter Pack

# How to start?

```
https://github.com/enotcode/flask-starter.git
cd flask-starter
pip install -r requirements.txt
python run.py
```
You will need to see this
```sh
* Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)
* Restarting with stat
* Debugger is active!
* Debugger PIN: 187-362-163
```
